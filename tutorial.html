🚀 Quick Start Guide

Welcome to DIRK Brain Portal - your AI agent orchestration platform!

Installation Steps

  1. Run the Setup Wizard
    python setup_wizard.py

    Follow the interactive prompts to configure your environment.

  2. Start the Platform
    ./start.sh # On Linux/Mac
    .\start.bat # On Windows

    This starts all services: Frontend, Backend API, and gRPC server.

  3. Access the Portal

    Open your browser and navigate to:

    Frontend: http://localhost:3000
    API Docs: http://localhost:3001/docs

First Time Setup

Configure Your First AI Provider:

  1. Click the Settings icon in the top-right corner
  2. Navigate to "AI Providers"
  3. Enter your API keys for Claude, GPT, or Gemini
  4. Click "Test Connection" to verify

🤖 Agent Management

Creating Agent Swarms

  1. Use the AI Prompt Box

    Located at the bottom of the interface. Enter natural language commands:

    "Create agents for e-commerce platform development"
  2. Select Templates

    Choose from pre-configured team templates:

    • Full-Stack Development Team
    • Data Analysis Pipeline
    • Security Audit Team
    • Creative Content Squad
  3. Deploy Agents

    Drag generated agents onto projects or use deployment commands:

    POST /api/pods/deploy
    {
    "agents": [...],
    "type": "swarm"
    }

  • SOC 2

Running Security Scans

Generate a Full Security Report:

curl -X POST http://localhost:3001/api/security/full-report \
-H "Content-Type: application/json" \
-d '{"project_id": "your_project"}'
⚠️ Security Best Practices:
  • Never commit API keys to version control
  • Use environment variables for sensitive data
  • Regularly update dependencies
  • Enable MrWolf validation for all deployments

🔍 Troubleshooting

Common Issues

Port Already in Use

Solution: Change ports in .env files or kill existing processes:

lsof -i :3000 # Find process
kill -9 [PID] # Kill process
gRPC Server Not Starting

Solution: Install gRPC dependencies:

pip install grpcio grpcio-tools
WebSocket Connection Failed

Solution: Check CORS settings and firewall rules.

Debugging Tips

  • Check logs in backend/logs/
  • Use browser DevTools Network tab for API issues
  • Run npm run dev with --verbose flag
  • Test individual services: python main.py
