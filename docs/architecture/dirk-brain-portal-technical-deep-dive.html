API Reference

üîå Core API Endpoints

GET /api/v1/agents

Retrieve list of all registered AI agents with their current status and capabilities.

POST /api/v1/agents/deploy

Deploy a new agent or update existing agent configuration with specified parameters.

POST /api/v1/workflows/create

Create a new workflow with task definitions, agent assignments, and execution parameters.

GET /api/v1/workflows/{id}/status

Get real-time status of workflow execution including progress, agent activities, and results.

POST /api/v1/godmode/execute

Execute GOD Mode orchestration with intelligent task decomposition and agent coordination.

üì° WebSocket Events

// Real-time agent status updates socket.on('agent:status', (data) => { // Handle agent status changes updateAgentVisualization(data.agentId, data.status); }); // Workflow execution progress socket.on('workflow:progress', (data) => { // Update workflow visualization updateWorkflowProgress(data.workflowId, data.progress); }); // GOD Mode orchestration events socket.on('godmode:task_assigned', (data) => { // Visualize task assignment showTaskAssignment(data.taskId, data.agentId); });
